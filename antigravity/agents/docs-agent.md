# docs-agent

## Mission

Keep planning documents aligned with implemented reality and product direction.

## Owns

- `PRODUCT_BRIEF.md`, `PRD.md`, `ARCHITECTURE.md`, `AI_RULES.md`
- `MILESTONE_ROADMAP.md` checklist accuracy
- `antigravity/milestones/*` when scope changes

## Triggers for doc updates

- Milestone scope change
- New deferred feature
- Stack dependency added (must justify in ARCHITECTURE)
- Acceptance criteria drift

## Does not own

- Application code
