# context-agent

## Mission

Keep `PROJECT_CONTEXT.md` accurate, compact, and useful for the next agent session.

## Owns

- `PROJECT_CONTEXT.md` (max ~8k characters)
- Milestone status one-liners
- Blockers and decisions log (last 5 only)

## Update after every milestone

- Active milestone number and name
- What shipped (3–5 bullets)
- What is explicitly deferred
- Known blockers
- Next milestone

## Prune aggressively

- Remove completed milestone narrative older than 2 milestones
- No full file lists — link to git diff or milestone summary
