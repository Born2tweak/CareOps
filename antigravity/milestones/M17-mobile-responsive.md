# M17 — Mobile Responsiveness

## Objective

Usable admin experience on tablet/phone for coordinators in the field (NFR accessibility polish).

## Scope

- Responsive queue table/cards
- Touch-friendly filters and row actions
- Nav collapses to drawer on small viewports
- WCAG 2.1 AA spot-check on shell + queue

## Out of scope

- Native apps
- Offline mode

## Acceptance criteria

- [ ] Queue usable at 390px width without horizontal scroll hell
- [ ] Primary actions reachable one-handed
- [ ] No critical contrast failures on main paths

## Tests

- Browser devtools: iPhone/Android sizes
- Manual coordinator walkthrough on phone

## Stop condition

Pilot users can triage exceptions on mobile browser.
